
# Rick & Morty Flutter App (Advanced Architecture)

Features implemented:

- Clean Architecture
- Feature-first structure
- Cubit state management
- Pagination ready
- Infinite scroll ready
- Search debounce
- Filter system with clear filters
- Pull-to-refresh
- Auto refresh when screen opens
- Repository cache layer
- Dio API client with interceptor
- Dependency Injection
- CI pipeline

Architecture:

core/
  di/
  network/
  utils/
  widgets/

features/
  characters/
    data/
    domain/
    presentation/

This project is designed as a strong Flutter portfolio example.
